# Sales Dashboard - .NET & Angular

Este proyecto es un Dashboard integral para la visualización y análisis de datos de ventas.

## Tecnologías
- **Backend:** ASP.NET Core Web API (.NET 8)
- **Frontend:** Angular 17+ con Tailwind CSS
- **Análisis:** Chart.js para visualización de datos dinámicos.

## Características
- Visualización de ventas por categoría en tiempo real.
- Cálculo de métricas clave (Ticket promedio, productos más vendidos).
- Interfaz moderna y responsive.

## Instalación
1. Clonar el repositorio.
2. En `SalesApi`: `dotnet run`.
3. En `SalesFrontend`: `npm install` y `npm start`.
